
# 🎯 Python Russian Roulette

심플 러시안 룰렛.

## ▶️ 실행

```bash
python main.py
```

## 🎮 게임방식

* 탄창 수, 총알 수, 참여자 수를 입력
* 턴마다 `1` (발사 예측) 또는 `0` (빈 칸 예측) 입력, 예측이 틀릴시 사망함
* 모두 사망하거나 `stop` 입력 시 게임 종료

## 📌 특징

* 랜덤 총알 배치
* 멀티 플레이 지원
* 사운드 효과 (`pygame` 사용)

## ⚠️ 요구 사항

* Python 3
* pygame (`pip install pygame`)
* `revol.mp3`, `none.mp3`, `fire.mp3` 파일 필요
